### 用素数性质简单判断

```
bool isPrime(int n){
    if(n <= 1) return false;
    int sqr = (int)sqrt(1.0 * n);
    for(int i = 2; i <= sqr; i++)
        if(n % i == 0) return false;
    return true;
}
```



### 打素数表

经典的埃氏筛法

```
const int maxn = 1e5;
int prime[maxn], pNum = 0;

void findPrime(){
    for(int i = 2; i < maxn; i++){
        if(prime[i] == 0){
            prime[pNum++] = i;
            for(int j = i * 2; j < maxn; j += i)
                prime[j] = 1;
            printf("%d\n", i);
        } 
    }

}
```

> 素数表的经典应用就是分解质因子，在有了素数表的情况下分解一个数字的因子的复杂度为O(n−−√)O(n)
