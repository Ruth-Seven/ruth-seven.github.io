## 斐波那契数列计算

<!-- more -->



三种思路：递归、dp和快速矩阵乘

**dp**

```
long long Fobonacci(unsigned n){
    if(n <= 0) return -1;
    long long fibN1 = 0;
    long long fibN2 = 1;
    if(n == 1) return fibN1;
    else if(n == 2) return fibN2;
    
    for(int k = 2; k <= n; k++){
		int temp = fibN2;
        fibN2 = fibN2 + fibN2;
        fibN1 = temp;
    }
    return fibN2;
}
```
