### 面21：调整数组使奇数位于偶数前面

### 题面：

如题

### 思路：

一可以使用插入排序的算法，不够复杂度有点高。

二可以使用双指针，交换奇偶数即可。

### 代码：

```
int ReorderOddEven(int *array, int length){
    int start = 0, end = length - 1, tmp = array[0];
    while(start < end){
        while(start < end && (array[end] & 1 == 0)) end--;
        array[start] = array[end];
        while(start < end && (array[end] & 1 == 1)) start++;
        array[end] = array[start];               
    }
    array[start] = tmp;    
}
```

### 思路扩展：

**解耦写法**

```
bool isEven(int p){
    return !(p & 1);    
}
bool isDivide3(int p){
    return  (p % 3);
}
//提供接口
bool func(int p){
    return isDivide3(p);
}

void  ReorderOddEven(int *array, int length){
    int start = 0, end = length - 1, tmp = array[0];
    while(start < end){
        while(start < end && func(array[end])) end--;
        array[start] = array[end];
        while(start < end && !func(array[start])) start++;
        array[end] = array[start];               
    }
    array[start] = tmp;    
}
```
